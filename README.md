# 🎗️ Global Cancer Patient Analysis & Prediction
This project uses **Machine Learning (Logistic Regression)** to classify cancer severity levels (Low, Medium, High) based on a dataset of 50,000 patients.

## 👤 Project Team
- **Amr Farag Gamal**
- **Youssef Hamdy**

## 📊 Key Features
- **Data Analysis:** Exploration of risk factors like smoking, genetic risk, and pollution.
- **ML Model:** Logistic Regression with 99.9% accuracy.
- **Interactive Dashboard:** Built with **R Shiny** for real-time prediction.
